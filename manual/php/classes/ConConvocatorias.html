conConvocatorias.php : 12

Controlador HTTP para la gestión de convocatorias de reuniones.

Expone los endpoints de lectura y escritura de convocatorias y delega la lógica de negocio en ModConvocatorias. Su responsabilidad principal es validar el método HTTP, leer parámetros o JSON de entrada y devolver las respuestas en formato JSON usando la infraestructura de ControladorBase.
