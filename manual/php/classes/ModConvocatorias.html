modConvocatorias.php : 9

Modelo de acceso a datos y reglas de negocio de convocatorias.

Se encarga de validar el payload recibido desde el controlador, persistir la cabecera y el orden del día, reconstruir los DTO de salida y aplicar los cambios de estado de las convocatorias.

SQL_LISTADO_BASE

modConvocatorias.php : 25

Consulta base reutilizable para los listados de convocatorias.

private string SQL_LISTADO_BASE = "SELECT\r\n c.idConvocatoria,\r\n c.fecha,\r\n c.cancelada AS estado,\r\n l.nombre AS lugar,\r\n ca.anioInicio,\r\n ca.anioFin,\r\n pr.nombre AS redacta,\r\n pi.nombre AS inicia\r\n FROM convocatoria c\r\n INNER JOIN lugar l ON l.idLugar = c.idLugar\r\n INNER JOIN cursoAcademico ca ON ca.idCurso = c.idCurso\r\n INNER JOIN participantes pr ON pr.idParticipante = c.idProfesorRedactaActa\r\n INNER JOIN participantes pi ON pi.idParticipante = c.idProfesorIniciaReunion"

Se completa dinámicamente con filtros por estado y el orden final del listado según el caso de uso.
