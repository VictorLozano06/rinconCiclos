    Class SidebarProfesorComponent

    Sidebar lateral del perfil profesor.

    Carga las categorías reales del backend, las transforma en enlaces de navegación y añade accesos fijos como "Inicio". También decide si debe mostrarse el acceso al panel de coordinador según los roles detectados.

    • mapCategorias(

          prefijo: string,
          rutaPadre?: string,

      Convierte el árbol DTO del backend en elementos navegables del sidebar.

      Cada categoría recibe:

      • un icono visual
      • una ruta basada en slug
      • sus hijas ya transformadas
      • un estado deshabilitado si es una opción especial no navegable

      Parameters

      • categorias: CategoriaDto[]

        Categorías del nivel actual.

      • prefijo: string

        Prefijo base de ruta, por ejemplo /profesor.

      • rutaPadre: string = ''

        Ruta ya acumulada cuando estamos en una subcategoría.

      Returns CategoriaSidebar[]

      Árbol listo para pintar en la plantilla.
